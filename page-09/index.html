특정 플랫폼 전용 기능 딥코 스튜디오 전용 앱빌더 UI ○ × 예선에서만 차별화. 본선 제한 시 범용 코딩 환경으로 전환 — 사전 대비 완료
설계 원리: 예선 문제를 설계할 때 특정 플랫폼 과도 의존 기능을 미션 핵심으로 삼지 않고, 자율주행·AI 비전·센서 제어 등 범용 로봇 기능 기반으로 구성함. 따라서 본선에서 고유 기술 일부가 제한되더라도 참가 학생의 역량 손실 없이 대회 참여 가능. 예선 문제는 재단 전문가 피드백 후 '26.4월 중 최종 확정 예정이며, 확정 후 세부 조정 가능함을 전제로 함.
▸ 재단·타 수행기관 협력을 위한 유연성
권역 조율
타 수행기관과 겹칠 경우 우리 강점 권역 중심으로 하되 재배분에 유연 대응. 모든 권역에 파트너·강사 네트워크 확보 완료 — 어떤 조합에서도 운영 품질 보장.
시기 조율
4~9월 예선 일정을 상충 없이 조정. 물류 순환 모델 기반 설계로 일정 변경 시 이동 경로만 조정하면 되는 유연한 물류 구조.
본선 규칙 수용
본선에서 특정 기업 고유 기술 제한 시 즉시 수용. 범용 기능 기반으로 본선 교육 커리큘럼 조정 가능.
