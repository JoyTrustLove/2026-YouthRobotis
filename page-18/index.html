▸ 본선 사전 멘토링 프로그램 (4주)
주차 방식 내용 담당 산출물
1주 온라인 본선 규칙 안내, 팀 구성 확정, 전략 수립 워크숍 (초등: 미션 분석 / 중학: 코스 분석 / 고교: 프로젝트 기획) 중앙대 멘토단 팀 전략서(초안)
2주 온라인 본선 심화 기능 교육 (초등: 그리퍼 제어 / 중학: PID 튜닝 / 고교: 딥러닝 모델 최적화), 1차 코드 리뷰 핸즈온 강사진
중앙대 멘토단		 코드 리뷰 피드백
3주 온라인 모의 경기 시뮬레이션 (영상 기반), 멘토 1:1 피드백, 전략 수정 중앙대 멘토단 모의 경기 기록, 수정 전략서
4주 대면 본선 장소 사전 방문(가능 시), 실제 아레나 리허설, 최종 점검·격려, 장비 최종 세팅 전 기관 공동 최종 체크리스트, 리허설 영상
▸ 데이터 연속성 관리 체계
예선 데이터 수집
성적 데이터: 트랙별 점수, 순위, 시도 횟수, 미션별 성공/실패
교육 데이터: 교육 이수 여부, 차시별 실습 완성도
만족도 데이터: 5점 척도 설문(교육 만족, 대회 만족, 재참여 의향)
인식 변화: 사전-사후 AI·로봇 인식 변화 조사 (5문항)
본선 활용
팀 매칭: 예선 성적 + 희망 역할 기반 최적 팀 구성
멘토링 맞춤화: 예선 취약 영역 기반 멘토링 집중 포인트 설정
성과 비교: 예선→본선 성적 향상도 측정 (교육 효과 입증)
최종 보고: 전체 참가자 데이터 기반 성과보고서 작성
본선 대회 지원 핵심 원칙
① 교육 연속성: 예선 교육에서 다루지 않은 본선 전용 기능(PID, 딥러닝 등)을 멘토링 2주차에 집중 교육하여 "교육 → 대회" 흐름을 끊기지 않게 함.
② 공정성 담보: 모든 팀에 동일 사양의 심화 키트를 무상 지급하여 장비 격차를 원천 차단.
③ 정서적 지원: 멘토가 기술 지도뿐 아니라 팀워크·발표 자신감까지 코칭하여 본선 현장 퍼포먼스 극대화.
▸ 본선 진출 팀 지원 패키지 구성: 심화 키트(추가 센서·부품) + 멘토링 4주 + 본선 교통비 지원(지방팀) + 숙박 지원(원거리 팀) + 본선 참가 확인서·수상 인증서 발급. 상세 지원 항목 및 예산은 면수 22(예산 계획)에 명시.
